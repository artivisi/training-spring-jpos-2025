About This Application

This is an educational ATM simulator that demonstrates ISO-8583 banking transaction processing. It simulates real ATM operations including balance inquiries, withdrawals, and cryptographic key management.

The simulator communicates with a backend server using the ISO-8583 standard protocol, which is widely used in the banking industry for electronic transactions.
